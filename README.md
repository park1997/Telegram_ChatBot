# 기능 소개

1. 산업시스템공학과, 건설환경공학과, 건축공학과, 기계로봇에너지공학과, 멀티미디어공학과, 융합에너지신소재공학과, 전자전기공학부,컴퓨터공학과,화공생명공학과,건축학과, 추가기능을 선택 할 수 있다.

2. 학과를 선택하면 이수체계도 및 졸업기준표 조회, 커리어넷 학과정보, 학과 공지사항, 선 이수과목 조회, 과목정보 조회, 졸업학점 계산기 선택이 가능하다.

3. 이수체계도 : 해당하는 학과의 이수체계도 사진



3. 각 선택지들은 파이썬코드를 통해 알고리즘을 만들었으며, 동국대학교 공과대학 졸업요건을 확인하기위한 수단으로 도움을 줄 수있다.
